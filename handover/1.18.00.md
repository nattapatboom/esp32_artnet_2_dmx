## 1. การจัดระเบียบโครงสร้าง JavaScript ใน web_pages.h (JS Code Organization)

โค้ด JavaScript ความยาวกว่า 1,600 บรรทัดใน [web_pages.h](./include/web_pages.h) ได้รับการแบ่งส่วนโครงสร้างด้วยแบนเนอร์คอมเมนต์ขนาดใหญ่อย่างเป็นหมวดหมู่ 6 ส่วนหลัก ดังนี้:
1. **SECTION 1: GLOBAL STATE & UI TABS:** จัดเก็บตัวแปรสถานะส่วนกลาง การตั้งค่า Save state และตัวควบคุมแท็บ
2. **SECTION 2: DIAGNOSTICS & TELEMETRY:** ฟังก์ชันดึงสถานะระบบและคำสั่งสแกนบัส I2C
3. **SECTION 3: SYSTEM SETTINGS MANAGEMENT:** ตัวจัดการข้อมูลตั้งค่าเครือข่าย การบันทึกค่าลงบอร์ด และการค้นหา Wi-Fi
4. **SECTION 4: OUTPUT CHANNELS MANAGER:** ตัวขับเคลื่อนการแมป GPIO, คำนวณคะแนนทรัพยากร, เครื่องมือทดสอบเอาต์พุต (Output Test Tools)
5. **SECTION 5: ESP-NOW PEERS CONFIGURATION:** ส่วนแสดงและตั้งค่าที่อยู่ MAC ของบอร์ดปลายทาง ESP-NOW
6. **SECTION 6: OTA FIRMWARE UPLOAD:** ระบบอัปโหลดและส่งคำสั่งเพื่ออัปเกรดเฟิร์มแวร์ผ่านเครือข่าย
