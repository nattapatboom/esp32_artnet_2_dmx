## 4. Function Generator (Type 16) — ตัวสร้างรูปคลื่น

- **คลาสใหม่:** `include/funcgen_control.h` — `FuncGenController`
- **ฮาร์ดแวร์:** ESP Timer + LEDC (ไม่ใช้ RMT เพราะ RMT loop mode ไม่เหมาะกับความถี่ช่วงกว้าง 1Hz-10kHz+)
- **DMX 5 Bytes:**
  - Byte 0-1: Frequency (16-bit little-endian, 1-10000 Hz)
  - Byte 2: Waveform type (0=Sine, 1=Triangle, 2=Sawtooth, 3=Square, 4=PWM)
  - Byte 3: Amplitude (0-255, scaled to LEDC duty 0-8191)
  - Byte 4: DC Offset (0-255, scaled to LEDC duty)
- **Adaptive Sample Rate:** 32 samples at high freq → 64 at medium → 128 at low freq for smooth waveform
- **Timer Callback:** ใช้ `esp_timer` (high-priority task context) → เรียก `ledcWrite()` ได้ปลอดภัย
- **scoreForType(16):** `3.0f` (ใช้ LEDC 1 channel)
- **Web UI:** Dropdown "Function Gen", Pin Selector (available GPIO), 5-ch DMX count, Test Panel

## 5. 7-Segment Direct Drive GPIO Mode

- เพิ่ม `mc_mode`:
  - `0` = TM1637 I2C-like (CLK+DIO 2 pins)
  - `1` = 74HC595 Shift Register (DATA+LATCH+CLK 3 pins)
  - `2` = 7-pin direct drive (consecutive GPIOs a/b/c/d/e/f/g + DP)
  - `3` = 8-pin direct drive (same as 7-pin + DP on pin2)
- **GPIO Mapping:** `pin`=segment A → consecutive pins A,B,C,D,E,F,G,(DP) starting from `ch.pin`

## 6. I2C / Display Disable Toggle

- **I2C Toggle:** `#i2c_enabled` checkbox → ซ่อน SDA/SCL/Speed fields → save: `i2c_sda=255, i2c_scl=255` (disable)
- **Display Toggle:** `#disp_enable_ck` checkbox → ซ่อน display config fields → save: `display_enabled=0` (force off)
- **loadSettings:** ใช้ค่า `i2c_sda!==255 && i2c_scl!==255` และ `display_enabled!==0` ในการ set checkbox state เริ่มต้น
