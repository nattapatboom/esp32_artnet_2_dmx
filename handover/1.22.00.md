## 1. Type 15 PWM DAC

- **scoreForType(15):** `2.0f`
- **outputTypeName(15):** `"PWM DAC"`
- **channelScore(15):** `2` (JS, for Web UI)
- **DMX:** 1 byte (8-bit) or 2 bytes (10/12/14/16-bit resolution)
- **Setup:** `motion_control.h` — `allocateLedc()` + `ledcSetup(pwmChan, ch.mc_freq, ledcResolution(ch))`
- **Update:** `ledcWrite(ch.dmxPort, val)` (same as Type 4)
- **Save:** `output_control.h` — saves `mc_freq` and `mc_resolution` for type 15
- **Web UI:** Dropdown "PWM DAC", GPIO-only (no expander), Resolution + PWM Carrier Frequency + RC Filter Ref

## 2. PWM Carrier Frequency for Function Generator

- `funcgen_control.h::begin(pin, ledcChan, pwmFreq)` — `pwmFreq` default 50000Hz
- `motion_control.h` — passes `ch.mc_freq ? ch.mc_freq : 50000` on setup
- `output_control.h` — saves `mc_freq` for type 16

## 3. RC Low-Pass Filter Calculator (Web UI)

- แสดงใน Section 6 เมื่อ type === 15 หรือ 16
- Input fields: R (Ω), C (nF)
- Auto-calculate: `fc = 1 / (2π × R × C)`
- Hint: Set PWM Carrier Frequency above to 3-5x fc
